# GoLangCC-4A



Remplir le .env

Dockerisation:

Afin de lancer le programme, veuillez lancer la commande suivante:

``docker build -t golangcc-4a .``
``docker run -it golangcc-4a``

Lancer le Dockerfile, puis lancer le programme avec la commande suivante:

``go run main.go``
        ou
``./main``
